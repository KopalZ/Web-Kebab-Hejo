1. Login ke Dashboard

URL Admin Dashboard

https://admin-dashboard-alpha-sage-15.vercel.app

Langkah-langkah:

  1. Buka URL admin di browser (Chrome/HP/PC)
  2. Kamu akan langsung diarahkan ke halaman Login
  3. Masukkan data berikut:
    • Email/Username: admin
    • Password: password123
  4. Klik tombol "Masuk"
  5. Kalau berhasil → masuk ke halaman Dashboard
  6. Kalau gagal → muncul pesan error merah (cek username & password)
Tips: Password default bisa diganti nanti via database. Hubungi developer jika ingin ganti password.

3. Hero Page — Banner & Link WhatsApp

Fungsi: Mengatur slide banner yang tampil di halaman utama website, dan mengubah nomor WhatsApp untuk tombol "Gabung Franchise".

A. Mengubah Link WhatsApp Franchise

  1. Di halaman Hero, cari bagian "Tombol Franchise"
  2. Lihat input yang berisi link WhatsApp saat ini
  3. Ubah nomor sesuai keinginan (contoh: https://wa.me/6281289222234)
  4. Klik "Simpan"
  5. Perubahan langsung terlihat di landing page setelah di-refresh

B. Menambah Slide Banner Baru

  1. Di bagian "Tambah Slide Baru", isi:
    • Path gambar: nama file gambar (contoh: banner-promo.jpg)
    • Catatan: keterangan slide (opsional)
  2. Klik "Tambah Slide"
  3. Slide baru akan muncul di landing page sebagai carousel banner
Catatan: File gambar harus sudah ada di folder public/ atau sudah diupload. API hanya menyimpan nama file, bukan file gambarnya.

C. Menghapus Slide

  1. Lihat tabel "Daftar Slide Aktif"
  2. Cari slide yang ingin dihapus
  3. Klik tombol "Hapus"
  4. Slide langsung hilang dari landing page

4. Katalog Menu — Atur Produk

Fungsi: Menambah, mengedit, dan menghapus produk menu yang tampil di landing page.

Filter Kategori

Klik tombol kategori (Semua, Kebab Pot, Kebab, Burger, HotDog, Pizza, Canai) untuk menyaring produk.

A. Tambah Menu Baru

  1. Klik tombol "Tambah Menu" di bagian atas
  2. Modal (popup) akan muncul dengan form:
    • Nama menu — wajib diisi
    • Harga — wajib diisi (angka, tanpa titik/koma)
    • Kategori — wajib pilih dari dropdown
    • Path gambar — opsional (contoh: uploads/menu-baru.jpg)
  3. Klik "Tambah"
  4. Produk baru langsung muncul di grid dan juga di landing page

B. Edit Menu

  1. Cari produk yang ingin diedit
  2. Klik tombol "Edit" pada kartu produk
  3. Kartu berubah menjadi form:
    • Ubah nama di kolom teks
    • Ubah harga di kolom angka
    • Ubah kategori dari dropdown
    • Ubah gambar di kolom path
  4. Klik "Simpan" untuk menyimpan
  5. Klik "Batal" jika tidak jadi mengubah

C. Hapus Menu

  1. Cari produk yang ingin dihapus
  2. Klik tombol "Hapus" pada kartu produk
  3. Konfirmasi: akan muncul popup Hapus "nama produk"?
  4. Klik "OK" → produk langsung terhapus
  5. Klik "Cancel" jika tidak jadi
Tips: Setiap perubahan di admin langsung tersimpan di database. Landing page akan menampilkan data terbaru setelah di-refresh (F5).

5. Jangkauan — Kelola Cabang

Fungsi: Mengelola daftar kota dan cabang outlet Grand Kebab Hejo beserta link Google Maps.

A. Tambah Cabang Baru

  1. Di halaman Jangkauan, isi form:
    • Kota: pilih kota dari dropdown
    • Nama Cabang: nama outlet (contoh: Grand Kebab Cikarang)
    • Link Google Maps: URL lokasi di Google Maps
  2. Klik "Tambah Cabang"
  3. Cabang baru langsung muncul di tabel dan landing page

B. Hapus Cabang

  1. Cari cabang di tabel
  2. Klik tombol "Hapus" di baris yang sesuai
  3. Cabang langsung terhapus dari database
Mendapatkan Link Google Maps:
  1. Buka Google Maps
  2. Cari lokasi cabang
  3. Klik "Bagikan" → "Salin link"
  4. Tempel link di form

6. Galeri — Atur Foto

Fungsi: Mengelola foto-foto galeri yang tampil di landing page.

A. Upload Foto Baru

  1. Klik area upload atau pilih file gambar dari komputer
  2. Bisa pilih beberapa file sekaligus
  3. Preview foto akan muncul dalam grid
  4. Tiap foto bisa dikasih catatan (keterangan)
  5. Klik "Upload Semua Sekarang"
  6. Semua foto akan masuk ke database
Penting! File gambar harus dicopy manual ke folder admin-dashboard/public/ sebelum upload. Hubungi developer untuk bantuan upload file.

B. Hapus Foto

  1. Lihat tabel daftar foto yang sudah terupload
  2. Klik "Hapus" di baris yang sesuai
  3. Foto langsung terhapus dari database

7. Logout

  1. Klik tombol "Keluar" di pojok kanan atas
  2. Konfirmasi: Apakah Anda yakin ingin keluar?
  3. Klik "OK" → kembali ke halaman Login
  4. Sesi login akan hilang saat tab browser ditutup

Tips Keamanan

  • Selalu logout setelah selesai menggunakan dashboard
  • Jangan simpan password di browser untuk akun admin
  • Ganti password default sesegera mungkin
  • Gunakan di perangkat yang aman dan jaringan terpercaya
